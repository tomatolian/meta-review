# meta-review
Python 3.10.11

## setup
```
git clone git@github.com:tomatolian/meta-review.git
cd meta-review
python -m venv env
source env/bin/activate
pip install -r requirements.txt
touch .env
```
`.env`の中身は鈴木まで

## アプリ起動
```
source env/bin/activate
streamlit run app.py
```
